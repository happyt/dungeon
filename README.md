# dungeon v.0.0.1

## Quick start

This was an app created to explore a hack evening idea of a dungeon maze. It's very simple, unfinished and just has simple N,S,E,W commands to move around rooms. 

To run this app

* Install node.js
* It should need no dependencies (v0.0.1)
* Run 'node cmd.js'
